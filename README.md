# Gochi 🐾

A kids-friendly terminal pet. Feed it, play with it, put it to sleep — and don't let it down!

## Requirements

- Node.js 18+

## Setup & run

```bash
npm install
npm start
```

## How to play

On each turn pick an action from the menu:

| Action | Effect |
|--------|--------|
| Feed   | Restores hunger |
| Play   | Boosts happiness (costs a little energy) |
| Sleep  | Restores energy |
| Clean  | Improves health |

Your pet's stats decay over time — even while the app is closed. Check in often!

Progress is saved automatically to `pet-save.json`.
